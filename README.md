# 环境安装
1. 安装conda环境
conda create -n mindspore python=3.9
conda activate mindspore

2. 安装华为开源自研AI框架MindSpore
pip install mindspore

1. Paillier同态加密库
pip install phe

1. 运行
python MindSpore.py
